# SteamSearcher

Userscript для поиска игр без русского языка на странице поиска Steam с автоматической проверкой наличия фанатских русификаторов на [ZoneOfGames.ru](https://www.zoneofgames.ru).

## Возможности

- **Поиск игр без русского языка** — сканирует страницу поиска Steam и находит все игры, не поддерживающие русский язык
- **Проверка русификаторов на ZOG** — для каждой найденной игры автоматически проверяет наличие фанатского перевода на ZoneOfGames.ru
- **Анти-Cloudflare защита** — автоматически определяет блокировки Cloudflare и повторяет запросы с нарастающей задержкой
- **Умное кэширование** — результаты запросов кэшируются в localStorage; ошибочные статусы не кэшируются, чтобы не закрывать путь к повторной проверке
- **Fuzzy matching** — нечёткий поиск названия игры на ZOG с помощью расстояния Левенштейна, корректная обработка артиклей (The, A, An)
- **Модальное окно** — все найденные игры отображаются в удобном модальном окне поверх страницы Steam с адаптивной сеткой карточек
- **Управление сканированием** — кнопки остановки, продолжения, перепроверки неудачных результатов и закрытия
- **Горячие клавиши** — быстрый запуск сканирования сочетанием `Shift + F`
- **Автопрокрутка** — скрипт автоматически прокручивает страницу Steam вниз для загрузки новых результатов
- **Обработка Rate Limit** — при получении ответа 429 от Steam API скрипт автоматически повторяет запрос с задержкой

## Установка

1. Установите расширение [Tampermonkey](https://www.tampermonkey.net/) для вашего браузера
2. Откройте [SteamSearcher.user.js](https://github.com/Onzis/SteamSearcher/raw/refs/heads/main/SteamSearcher.user.js) — Tampermonkey предложит установить скрипт
3. Подтвердите установку

## Использование

1. Откройте страницу поиска Steam: `https://store.steampowered.com/search/*`
2. В левом нижнем углу появится круглая кнопка с иконкой поиска
3. Нажмите на неё (или нажмите `Shift + F`) — откроется модальное окно и начнётся сканирование
4. Скрипт автоматически прокручивает страницу Steam вниз и проверяет каждую игру

### Кнопки управления

| Кнопка | Описание |
|--------|----------|
| Стоп | Приостановить сканирование |
| Продолжить | Продолжить сканирование после остановки |
| Перепроверить | Перепроверить только игры с ошибками и без русификатора (статусы «Не найдено на ZOG», «Нет русификатора», «Ошибка загрузки ZOG») |
| Закрыть | Закрыть модальное окно |

### Горячие клавиши

| Сочетание | Действие |
|-----------|----------|
| `Shift + F` | Запустить сканирование с начала |

### Статусы проверки ZOG

| Статус | Цвет | Описание |
|--------|------|----------|
| Русификатор есть | Зелёный | На ZOG найден фанатский перевод, отображается ссылка и список переводов |
| Нет русификатора | Оранжевый | Игра найдена на ZOG, но переводы отсутствуют; отображается ссылка на страницу игры |
| Не найдено на ZOG | Серый | Игра не найдена в каталоге ZoneOfGames |
| Ошибка загрузки ZOG | Красный | Не удалось загрузить страницу (Cloudflare, сетевая ошибка) |
| Проверка ZOG... | Голубой | Идёт проверка, анимированный спиннер |

## Технические детали

### API и источники данных

Скрипт использует несколько API для получения информации:

- **Steam Store API** (`store.steampowered.com/api/appdetails`) — проверка списка поддерживаемых языков для каждого приложения
- **Steam Browse API** (`api.steampowered.com/IStoreBrowseService/GetItems/v1`) — получение английского названия игры для более точного поиска на ZOG
- **ZoneOfGames** (`zoneofgames.ru`) — каталог фанатских переводов, скрапинг страниц с алфавитным индексом

### Задержки между запросами

| Источник | Задержка | Причина |
|----------|----------|---------|
| Steam API | 500 мс | Стандартная задержка между запросами |
| ZOG | 2 500 мс | Предотвращение блокировки сервером |
| Cloudflare | 8 000 мс + нарастающая | При повторных попытках: 8 с, 16 с (до 2 повторов) |
| Steam 429 | 2 500 мс | Задержка при Rate Limit, до 2 повторов |

### Кэширование

Результаты проверки хранятся в `localStorage` для исключения повторных запросов при повторном посещении страницы:

- Результаты проверки языка: ключ `lang_ru_v5_{appId}` (хранит `true`/`false`)
- Результаты проверки ZOG: ключ `zog_ru_v3_{appId}` (хранит JSON с детальной информацией)
- Статусы `not_found` и `error` **не кэшируются** — они могут быть вызваны временной блокировкой Cloudflare, и повторная проверка может дать другой результат
- Кэшируются только точные результаты: `found` (русификатор найден) и `no_translations` (игра есть на ZOG, но переводов нет)
- При обновлении версии скрипта старые ключи кэша (`zog_ru_v1_*`, `zog_ru_v2_*`) автоматически удаляются

### Поиск на ZOG

1. Получает английское название игры через Steam Browse API (если не удалось — используется название со страницы)
2. Определяет буквенный индекс: для английских названий используется английский алфавит, для русских — русский
3. Обрабатывает артикли (The, A, An): если название начинается с артикля, ищет также по второму слову
4. Загружает соответствующую страницу каталога ZoneOfGames через `GM_xmlhttpRequest`
5. Ищет ближайшие совпадения по расстоянию Левенштейна (порог сходства 45%)
6. Совпадения, начинающиеся с названия игры, приоритетнее, чем просто похожие
7. Переходит на страницу лучшего совпадения и проверяет секцию «Переводы»
8. Извлекает список всех доступных русификаторов с названиями и ссылками

### Обработка Cloudflare

При запросах к ZoneOfGames скрипт проверяет признаки Cloudflare-блокировки:

- HTTP-статусы 403 и 503
- Ключевые фразы в теле ответа: `cf-browser-verification`, `cf_chl_opt`, `Just a moment`, `Checking your browser`, `Attention Required`, `Cloudflare`

При обнаружении блокировки скрипт повторяет запрос с нарастающей задержкой (до 2 дополнительных попыток).

## Совместимость

- Chrome / Edge (с Tampermonkey)
- Firefox (с Tampermonkey)
- Требуется доступ к: `store.steampowered.com`, `api.steampowered.com`, `zoneofgames.ru`
- Требуется разрешение `GM_xmlhttpRequest` для кросс-доменных запросов к ZOG
